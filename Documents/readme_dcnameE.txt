The file "dcnameE," a comma delimited file, lists a complete history of all common stocks in the US from CRSP database since 1926.  You need a historical SIC code to identify to which industry a stock belongs in Year 2000, for example, because a firm might migrate to different industries over time due to corporation restructuring and mergers and acquisitions.

The data fields in the file "dcnameE" are described as follows:

KYPERMNO,NCUSIP,SICCD,SNAICS,NAMEDT,NAMEENDDT,SHRCD,TSYMBOL,COMNAM

The definition of these variables can be found in the stock files of CRSP from WRDS data service.
